# Product Hunt Launch Day Checklist

This skill provides a comprehensive, hour-by-hour checklist to execute your Product Hunt launch flawlessly.

## When to Use This Skill

- Final preparation night before launch
- During the 24-hour launch window
- Coordinating team activities on launch day
- Ensuring all critical actions are completed

## The Night Before (T-12 Hours)

### Final Checks
- [ ] Product Hunt listing is complete and scheduled
- [ ] All gallery images uploaded and ordered
- [ ] Video tested and working
- [ ] Maker comment pre-written and saved
- [ ] All links tested and working
- [ ] Website ready for traffic spike
- [ ] Analytics tracking verified
- [ ] Email sequences scheduled
- [ ] Social posts scheduled
- [ ] Supporter network reminded

### Personal Preparation
- [ ] Alarm set for launch time (if applicable to your timezone)
- [ ] Phone charged
- [ ] Computer ready
- [ ] Food/drinks prepared
- [ ] Team communication channel ready
- [ ] Get rest!

## Launch Hour (12:01 AM PST)

### Immediate Actions (First 10 Minutes)
- [ ] Verify product is live on Product Hunt
- [ ] Confirm listing appears correctly
- [ ] Post maker comment immediately
- [ ] Share PH link internally with team
- [ ] Screenshot for records
- [ ] First social media post goes out

### First Hour Focus
- [ ] Monitor for any issues
- [ ] Respond to first comments (< 5 minutes)
- [ ] Wave 1 of supporter outreach (close inner circle)
- [ ] Check ranking position
- [ ] Note early feedback

**Target:** 30-50 upvotes in first hour

## Early Hours (1 AM - 6 AM PST)

### Ongoing Tasks
- [ ] Continue responding to all comments
- [ ] Monitor for technical issues
- [ ] Wave 2 supporter activation (if planned)
- [ ] Track progress vs benchmarks
- [ ] Screenshot milestones

### Team Coordination
- [ ] Check in with team members
- [ ] Handoff if working in shifts
- [ ] Document any issues

**Target:** 100+ upvotes by 6 AM PST

## Morning (6 AM - 12 PM PST)

### Peak Activity
- [ ] Wave 3 supporter activation (US East Coast)
- [ ] Monitor social media mentions
- [ ] Respond to all new comments
- [ ] Share progress updates
- [ ] Wave 4 email sends
- [ ] LinkedIn/Twitter engagement

### Mid-Morning Check
- [ ] Current ranking position
- [ ] Upvote velocity
- [ ] Comment quality
- [ ] Any concerning patterns
- [ ] Competitor activity

**Target:** 200+ upvotes by noon PST

## Afternoon (12 PM - 6 PM PST)

### Sustained Push
- [ ] Wave 5 supporter activation
- [ ] Progress update post on social media
- [ ] Continue comment responses
- [ ] Retweet/share supporter posts
- [ ] Monitor and adjust strategy

### 4 PM PST Check
- [ ] Ranking trajectory analysis
- [ ] Final push needed?
- [ ] Activate remaining supporters
- [ ] Behind-the-scenes content

**Target:** 400+ upvotes for top 5

## Evening (6 PM - Midnight PST)

### Final Push
- [ ] Wave 6 outreach to non-responders
- [ ] Final social media push
- [ ] Last reminder to close supporters
- [ ] Continue comment engagement
- [ ] Thank early supporters

### 10 PM PST
- [ ] Assess final ranking position
- [ ] Begin preparing thank you messages
- [ ] Screenshot final stats
- [ ] Document learnings

## Post-Midnight (Day 2)

### Immediate After-Launch
- [ ] Final ranking recorded
- [ ] Thank you post on social media
- [ ] Thank you emails to supporters
- [ ] Team celebration/debrief
- [ ] Document all metrics

## Emergency Protocols

### If Website Goes Down
1. Don't panic
2. Check hosting/server status
3. Implement backup if available
4. Post update on PH comments
5. Keep engaging while fixing

### If Comments Turn Negative
1. Respond professionally
2. Acknowledge concerns
3. Provide solutions/clarifications
4. Don't get defensive
5. Thank for feedback

### If Ranking Drops Suddenly
1. Check for algorithm flags
2. Review recent activity patterns
3. Don't panic-activate supporters
4. Focus on quality engagement
5. Accept what you can't control

### If You Get Flagged/Unfeatured
1. Don't spam outreach
2. Contact PH support if legitimate
3. Review your practices
4. Learn for next time
